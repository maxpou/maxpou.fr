Let say I want to rebase the last 3 commits:

For massives features, you might end up with a branch with a few commits inside. And then tests are failing and you want to identify the ‚Äúguilty commit‚Äù. You can use rebase --exec to execute a command on each commit of the history.

ü¶ãStash

Because it‚Äôs not all about git stash and git stash pop ;)
